# Design note: the anti-hallucination triad for vertical RAG

(Outline — to be expanded once the hybrid-rag package lands in Phase 3.)

1. Multi-route retrieval: query embedding + LLM-decomposed atomic sub-questions + n-gram full-text — why semantic-only retrieval misses "bridge clauses" that are legally essential but semantically distant.
2. Authority-tier re-ranking: statute > regulation > local rule; encode in code AND prompt.
3. Citations parsed from retrieved source material by the backend — never generated by the model.
4. Canonical-answer override layer: human-verified answers injected as highest priority; above a similarity threshold, serve the canonical answer directly and skip the model entirely.
5. Prompt-position lesson: an override merely *retrieved* is not an override *obeyed* — placement at the end of the prompt and explicit conflict rules matter.
6. Guardrails: no rules the source text doesn't contain; opinions require a citable basis; cases illustrate, statutes decide.
